### Capstone Developed by [Cody Jones](https://github.com/cbjones93)
This is a simple web Api/React app that uses Firebase authentication.
Treasure Chest aims to help consumers with the computer parts shortage by creating a marketplace for used parts. In this app, users can navigate through posts by date, category, or sellers.

# Installation
1. Clone this repository
2. Run the two .SQL files in the SQL directory
